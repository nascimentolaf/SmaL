# Smart Light (SmaL)
This repository contains the receiver implementation of a low-cost multiplatform Visible Light Communication (VLC) receiver prototype. 
The project was developped in the year of 2022 as a undergraduate thesis for the degree of bachelor in electrical engineer of the first author.

## Meta
* **Authors**: Alaf Nascimento, Klaas Zwaag, Wesley Costa,  Higor Camporez, Francisco Santos, Jair Silva.
* **Affiliation**: Electrical Engineering Department, Federal University of Espírito Santo (UFES), Vitória/ES, Brazil
* **License**: [MIT](LICENSE)
* **Development Year**: 2022

## Features
* Manchester decoding
* BER evaluation
* Web Interface for data visualization
